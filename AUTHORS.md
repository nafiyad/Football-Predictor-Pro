## Project Information

**Football Predictor Pro** is an AI-powered football prediction system developed from scratch as a comprehensive machine learning project.

### Key Achievements
- 🎯 85.26% prediction accuracy on match outcomes
- 🔥 97.70% accuracy on high-confidence predictions
- 🤖 5 advanced ML models working in ensemble
- 🎨 Industry-level modern GUI
- 📊 1,296+ real matches analyzed

### Technologies Used
- **Language**: Python 3.9+
- **GUI**: CustomTkinter
- **Database**: SQLite with SQLAlchemy ORM
- **ML Libraries**: scikit-learn, XGBoost, LightGBM, TensorFlow
- **Data Processing**: pandas, numpy
- **API Integration**: football-data.org
